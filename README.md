# Bunny Life - Sistema de Normativas

Sistema completo de gestión de normativas para servidor de FiveM Roleplay, desarrollado con arquitectura modular que combina una interfaz web moderna con un backend robusto y un bot de Discord integrado.

## Descripción

Bunny Life es una plataforma web profesional diseñada para gestionar y visualizar normativas de un servidor de roleplay. El sistema permite a los administradores crear, editar y organizar contenido de manera dinámica, mientras que los usuarios pueden acceder a las normativas de forma pública y realizar búsquedas eficientes.
